ScrollAnimate.js

ScrollAnimate.js is a JavaScript library designed to facilitate seamless animations on web pages triggered by scroll events. It simplifies the process of animating elements based on their visibility in the viewport while providing flexibility in defining animation types, durations, delays, and easing modes through HTML data attributes. This library enables the initiation and control of animations as users scroll through the webpage, enhancing the user experience and adding dynamic visual effects to content.

Get started

Include the Files: Ensure you have both ScrollAnimate.js and ScrollAnimate.css files in your project directory. Link Files in HTML: In the <head> section of your HTML file, link the ScrollAnimate.css file to apply predefined styles for animations:

<link rel="stylesheet" href="ScrollAnimate.css">

Script Inclusion: Place the ScrollAnimate.js file before the closing </body> tag to ensure it loads after the DOM:

<script src="ScrollAnimate.js"></script>

Define Animated Section: Add the data attribute - data-animated-section to animate elements inside a section. You can add attributes like data-offset and data-play-once to set parameters to the block. Define Animated Elements: Utilize data attributes (data-animation, data-duration, data-delay, data-easing) to specify animation types, duration, delays, and easing modes.

<div data-animated-section>
    <div data-animation="fadeIn" data-duration="1s" data-delay="0s" data-easing="ease">Animated Element</div>
</div>

Fade in

data-animated-item data-animation="fadeIn"

Scale up

data-animated-item data-animation="scaleUp"

Slide left

data-animated-item data-animation="slideLeft"

Slide right

data-animated-item data-animation="slideRight"

Slide up

data-animated-item data-animation="slideUp"

Slide down

data-animated-item data-animation="slideDown"
